# Clone the project

```
git clone https://github.com/joseotaviopc/analu.git
```

## Install the dependencies

Just run the next command in a terminal (make sure that you have installed Yarn).

```
yarn
```

## Start the project

After the installation of the dependencies run:

```
yarn dev
```
